Minimalist Social Network
=============
Prototype v2
-------------

A social network for minimalists, implemented as a single page javascript application.
The technical manual and user manual were part of the coursework assignment.


Technologies used
-------------
*   AngularJS
*   PHP
*   HTML5
*   Sass
*   Bootstrap
*   Protractor
*   Jasmine
*   Grunt
