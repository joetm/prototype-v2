EDIT PROFILE

Prototype This page would allow the user to add more details to the profile.

The page would be identical to the user profile, but with additional fields to...
  • add a birthdate (which is currently NULL),
  • add an "about me" text,
  • upload or change an avatar picture, and
  • re-arrange the things in a different order.
The first three items in this list are repetitions of client-server interation through forms.
The fourth item has been implemented on the user profile.
